Create the connection to Ansible Tower using the following parameters:

```yaml
URL for REST API:      from Demo-UI
Authentication type:   User ID/Password
User:                  admin
Password:              from Demo-UI
```

1. Under `Ansible Automation Controller`, click on `Add connection`
1. Click `Connect`

1. üîé Get the data from your Demo-UI section  **Configure Runbooks - Ansible Automation Controller**



1. Fill out the fields:

	![image](./../images/29_runbook.png)

	```yaml
	URL for REST API:      from Demo-UI
	Authentication type:   User ID/Password
	User:                  admin
	Password:              from Demo-UI

1. Click `Done`

1. Make sure that the Connection Status turn green after a few seconds

## üöÄ TASK: Create a runbook

<a href="https://www.ibm.com/docs/en/cloud-paks/cloud-pak-watson-aiops/3.6.0?topic=library-create-runbook" target="_blank" >üåè Documentation for the Task: Creating a runbook</a>

Create a Runbook based on the `CP4WAIOPS Mitigate Robotshop Ratings Outage` Ansible Playbook.





<Accordion>
<AccordionItem title="üí° Need Help?">




1. Select tab `Runbooks`
1. Click `Create Runbook`
1. Name it `RobotShop Mitigate MySQL Problem`
1. Click `Add automated step`
1. Select `CP4WAIOPS Mitigate Robotshop Ratings Outage`
1. Click `Select this action`
1. Click `Mapping` / `Select`
1. Select `Fixed value`

	![image](./../images/31_runbooka.png)

1. üîé Get the data from the Demo-UI under the section `Configure Runbooks - Runbook Parameters` and copy the parameter Value. Make sure you copy everything between and including the curly brackets.
1. Paste the content in the Value field

	![image](./../images/32_runbooka.png)

1. Click `Save`
1. Click `Actions` and `Publish`

</AccordionItem>
</Accordion>
